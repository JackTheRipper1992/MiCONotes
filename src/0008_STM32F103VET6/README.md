# STM32F103VET6

## Tools

* [JLinker download](src/jlink.zip)  
这里需要注意的就是，之前安装MiCO的JLink驱动，导致在安装的时候遇到总是识别为libusb设备，即使卸载了，还是这样，处理办法是卸载了libusb之后，再卸载JLink驱动，重新安装对应的上面的Jlink驱动，载卸载libusb驱动，就可以了。
